# Assets Directory

This directory contains app assets. Replace the placeholder files with actual images:

- `icon.png` - App icon (1024x1024px recommended)
- `splash.png` - Splash screen image
- `adaptive-icon.png` - Android adaptive icon (1024x1024px)
- `favicon.png` - Web favicon (48x48px or larger)

**Note:** The placeholder files are empty. You need to add actual image files for the app to work properly in production.
