# React Native UI Library Recommendations

## 🎯 Best Options for Your Expo/React Native Project

Since you're using **Expo**, **NativeWind (Tailwind)**, and need **iOS, Android, and Web** support, here are the top recommendations:

---

## 1. **React Native Paper** ⭐ (Recommended)

**Why it's great:**
- ✅ Fully open source (MIT License)
- ✅ Works on iOS, Android, and Web
- ✅ Excellent Expo compatibility
- ✅ Material Design 3 components
- ✅ Built-in theming system
- ✅ TypeScript support
- ✅ Active maintenance (1M+ downloads/week)
- ✅ Great documentation

**Installation:**
```bash
npm install react-native-paper react-native-vector-icons
```

**Components included:**
- Button, TextInput, Card, Dialog, BottomSheet
- Checkbox, RadioButton, Switch, Slider
- Avatar, Badge, Chip, Divider
- List, Menu, Snackbar, Toast
- And 30+ more components

**GitHub:** https://github.com/callstack/react-native-paper

---

## 2. **Gluestack UI** ⭐ (Best for NativeWind)

**Why it's great:**
- ✅ Open source (Apache 2.0)
- ✅ Built specifically for NativeWind/Tailwind users
- ✅ Universal components (iOS, Android, Web)
- ✅ Styled System + Tailwind-like API
- ✅ Excellent performance
- ✅ TypeScript first
- ✅ Modern design system

**Installation:**
```bash
npm install @gluestack-ui/themed @gluestack-style/react
```

**Components included:**
- Button, Input, Card, Modal, Toast
- Checkbox, Radio, Switch, Slider
- Avatar, Badge, Progress, Spinner
- And more...

**GitHub:** https://github.com/gluestack/gluestack-ui

---

## 3. **NativeBase v3**

**Why it's great:**
- ✅ Open source (Apache 2.0)
- ✅ Works with NativeWind
- ✅ Cross-platform (iOS, Android, Web)
- ✅ Large component library
- ✅ Good theming support

**Installation:**
```bash
npm install native-base react-native-svg react-native-safe-area-context
```

**GitHub:** https://github.com/GeekyAnts/NativeBase

---

## 4. **Tamagui**

**Why it's great:**
- ✅ Open source (MIT License)
- ✅ Extremely high performance
- ✅ Universal (iOS, Android, Web)
- ✅ Optimized animations
- ✅ Great for complex UIs

**Installation:**
```bash
npm install @tamagui/core @tamagui/config
```

**Note:** More complex setup, but excellent performance

**GitHub:** https://github.com/tamagui/tamagui

---

## 5. **React Native Elements**

**Why it's great:**
- ✅ Open source (MIT License)
- ✅ Very popular (20k+ stars)
- ✅ Cross-platform
- ✅ Easy to customize
- ✅ Good community support

**Installation:**
```bash
npm install react-native-elements react-native-vector-icons
```

**GitHub:** https://github.com/react-native-elements/react-native-elements
